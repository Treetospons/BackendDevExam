
# BackendDevExam



## Installation

Install my-project with npm

```bash
  npm install
```
    
Install my-project with yarn

```bash
  yarn install
```
## Run Locally

run with npm 

```bash
  npm run start
```

run with npm run dev

```bash
  npm run dev
```

run with yarn 

```bash
  yarn start
```

run with yarn run dev

```bash
  yarn dev
```
